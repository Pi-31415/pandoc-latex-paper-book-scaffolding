Project Structure ​

Currently, when we open the Unity Project, we can see a well organized folder structure.

The simulation design is modular. All the scripts are in the Scripts Folder and is heavily documented. Below are a summary of all the scripts used.

Script NamePurposeFunctionality Description
updateMeshCollider_NewDynamically updates the mesh colliderSyncs mesh collider with blend shape deformations to accurately reflect the mesh's current visual state.
VRLocationAdjusterControls Player movement using Oculus inputsFacilitates the syringe calibration VR space
RemoveCollidersEditorProvides a tool for removing collidersAdds a tool in the Unity Editor for managing collider components during development, useful in haptic feedback adjustments.
PersistentTrailVisualizes the trajectory of the Haply DeviceEnhances debugging by displaying and saving the motion trajectory of Haply arm.
ChairHeightAdjusterAdjusts height of Player in VR WorldAllows ergonomic alignment adjustments within the VR setting on how high the OVRCamera component is.
AssignPhysicsMaterialEditorAssigns physics materials to collidersAutomates the setup Hard and Soft physical materials for haptic integration.
PassThroughControllerManages Oculus Passthrough featureToggles passthrough visibility dynamically,used for syringe calibration..
MainThreadDispatcherEnsures thread-safe execution on main threadMaintains stability in networked or complex simulations by safely managing actions affecting game objects. This is required for syringe to Unity communication.
CombinedControllerProcesses sensor data for syringe controlSyncs the orientation of the virtual syringe with its physical counterpart.
LaserPositionManages a VR laser pointerProvides a visual tool for pointing and selection, enhancing interaction within the VR environment.
UDPSerialReceiverHandles UDP data receptionConnects Unity with external haptic devices (syringe, cheek retraction module) for real-time interaction control.
FaceControlAnimates facial expressionsAdjusts facial blend shapes based on inputs, namely the cheek retraction module.
DrawLineDraws a line between two pointsUsed for visualizations like indicating paths or connections for debugging purposes.
HapticsToggleManagerManages haptic feedback collidersIt goes through all the Haptic enabled game objects and toggles the mesh colliders.
